This project demonstrates dynamic lighting effects using addressable LEDs (WS2812/NeoPixel) controlled by an Arduino Nano. The LEDs are connected to digital pin D6, and the system generates a variety of visually appealing patterns and color transitions.

🔧 Hardware Used
Arduino Nano
WS2812 / NeoPixel LED strip (10 LEDs)
Power supply (5V)
Connecting wires
⚙️ Features & Effects
✅ Individual LED control
✅ Green LEDs lighting up one by one
✅ Smooth color transitions
✅ Looping animations
✅ Pink fade in and fade out effect
✅ Multiple pattern modes
✅ Bidirectional movement:
Red light moves from left → right
Then right → left
✅ Color-changing edges:
Changes color when reaching either end
✅ 50+ color variations
✅ Multi-LED activation:
3 LEDs glow simultaneously
🎨 Patterns Included
Sequential lighting
Color wipe
Fade effects

Multi-color cycling
Edge-triggered color switching
🚀 How It Works

The Arduino Nano sends data signals through pin D6 to control each LED individually. Using timing-based loops and color arrays, the program creates smooth animations and pattern transitions across the LED strip.
